# LRCG (LDraw Rules Computer Graphics)

### Author

Richard Bruce Baxter - Copyright (c) 2006 Baxter AI (baxterai.com)

### License

* None

### Description

Procedural LDraw terrain generation software (April 2006)

* Matlab Terrain Data Generator (XY mesh)
* C++ Convert Mesh To Ldraw
